Amethystia API v0
Documentation of the /api/v0 route

Functions:

  • /checkversion/<user_version> - Get latest version of modpack and compare it to client's reported version.
    Responses:
    • if <user_version> is equal to server's value, server will respond with it's value: 0.1.0-alpha
    • if <user_version> is not equal to the server's value, server will respond with it's value and notice user to update the modpack: 0.1.0-alpha. Consider updating modpack!
